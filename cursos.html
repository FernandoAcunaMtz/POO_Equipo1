    Desarrollo UI/UX

    ¿De qué trata este curso?
    Aprende a diseñar interfaces de usuario atractivas y experiencias de usuario efectivas directamente desde tu editor de código. Este curso combina teoría del diseño con implementación práctica usando VS Code, extensiones para diseño y principios de usabilidad aplicables al desarrollo frontend.

    Contenido del curso:
    • Semana 1: Fundamentos de UI/UX para desarrolladores
    • Semana 2: Herramientas de diseño en VS Code (extensiones, emmet, snippets)
    • Semana 3: Prototipado rápido con HTML/CSS en VS Code
    • Semana 4: Pruebas de usabilidad y mejora de interfaces
    • Semana 5: Integración con herramientas de diseño (Live Server, Prepros)

    ¿A quién va dirigido?
    A desarrolladores frontend, full-stack o estudiantes que quieran mejorar sus habilidades de diseño de interfaces directamente en su entorno de desarrollo.

    Requisitos:
    • VS Code instalado en tu computadora
    • Conocimientos básicos de HTML y CSS
    • Acceso a internet
    • (Opcional) Extensión "Live Server" para VS Code

    Beneficios adicionales:
    • Flujos de trabajo eficientes para diseño y desarrollo
    • Integración de herramientas de diseño en tu entorno de codificación
    • Desarrollo de interfaces responsivas desde el primer día

    Minijuegos con Python

    ¿De qué trata este curso?
    Aprende a programar de forma divertida creando minijuegos clásicos con Python. Desde juegos de consola simples hasta proyectos gráficos básicos con Pygame, este curso te guiará paso a paso en el desarrollo de juegos mientras aprendes fundamentos de programación.

    Fundamentos de Bases de Datos con SQL

    ¿De qué trata este curso?
    Domina los conceptos esenciales de las bases de datos relacionales y aprende a trabajar con SQL usando MySQL. Desde instalación y configuración hasta consultas complejas, este curso te proporcionará las habilidades fundamentales para interactuar con bases de datos en entornos profesionales.

    Contenido del curso:
    • Semana 1: Introducción a bases de datos y MySQL (instalación y configuración)
    • Semana 2: Diseño de bases de datos (modelado ER, normalización)
    • Semana 3: Consultas SQL básicas (SELECT, INSERT, UPDATE, DELETE)
    • Semana 4: Consultas avanzadas (JOINs, subconsultas, funciones agregadas)
    • Semana 5: Administración básica (usuarios, permisos, optimización)

    ¿A quién va dirigido?
    A principiantes en el mundo de las bases de datos, desarrolladores que necesitan aprender SQL, analistas de datos, administradores de sistemas que requieren conocimientos básicos de MySQL, y cualquier profesional que trabaje con información estructurada. Requisitos:
    • Computadora con Windows, Mac o Linux
    • Conexión a internet para descargar software
    • No se requieren conocimientos previos de bases de datos
    • Conocimientos básicos de informática

    Beneficios adicionales:
    • Acceso a scripts SQL de ejemplo y ejercicios prácticos
    • Instrucciones detalladas para instalar MySQL en diferentes sistemas operativos
    • Casos de estudio con bases de datos reales
    • Certificado de finalización al completar el curso
    • Guía de recursos para profundizar en temas avanzados

    Piensa como programador

    ¿De qué trata este curso?
    Desarrolla el pensamiento lógico y algorítmico esencial para programar, sin enfocarte en un lenguaje específico. Aprenderás a descomponer problemas, diseñar soluciones eficientes y entender los conceptos fundamentales que todo programador utiliza, mediante ejercicios prácticos y ejemplos de la vida real.

    Contenido del curso:
    • Semana 1: Fundamentos del pensamiento computacional (descomposición, patrones, abstracción)
    • Semana 2: Lógica básica de programación (variables, condicionales, bucles)
    • Semana 3: Estructuras de datos simples (listas, diccionarios, conjuntos)
    • Semana 4: Algoritmos comunes (búsqueda, ordenamiento, recursión)
    • Semana 5: Resolución de problemas complejos (estrategias, debugging, optimización)

    ¿A quién va dirigido?
    A principiantes absolutos en programación, estudiantes de carreras técnicas, profesionales de otras áreas que necesitan entender lógica computacional, y cualquier persona interesada en desarrollar habilidades de resolución de problemas sistemáticos. Requisitos:
    • No se requiere experiencia previa en programación
    • Computadora o tablet para los ejercicios prácticos
    • Mentalidad abierta y ganas de aprender
    • Conocimientos básicos de operar una computadora

    Beneficios adicionales:
    • Ejercicios prácticos basados en situaciones reales
    • Acceso a plataforma con simuladores visuales
    • Certificado de finalización
    • Guía de próximos pasos para aprender a programar
    • Comunidad de estudiantes para resolver dudas
